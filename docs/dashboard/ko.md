# 대시보드 사용법

[English](./)

로그인된 대시보드에서 NAS로 영상, 오디오, 자막을 개별 다운로드하고 저장된 파일을 관리할 수 있습니다.

## 새 다운로드

1. **Video**, **Audio**, **Subtitle** 중 하나를 선택합니다.
2. 화질 또는 자막 언어를 선택합니다.
3. 지원되는 URL을 붙여 넣고 **Submit**을 누릅니다.

## 현재 작업

Current Activity에는 현재 제목, 채널, 썸네일, 진행률, 전송 속도와 남은 시간이 표시됩니다. **Up next**에서는 대기 중인 요청의 처리 순서, 요청 출처와 선택 화질을 확인할 수 있습니다.

## 파일과 기록

- **List**와 **Grid** 보기를 전환할 수 있으며 선택한 보기는 브라우저에 저장됩니다.
- 제목, 채널, 파일명과 메타데이터 상태를 검색할 수 있습니다.
- 상태와 미디어 종류로 필터링할 수 있고 기본 정렬은 최신 다운로드 순입니다.
- 결과는 20개씩 번호가 있는 페이지로 나뉩니다.
- 항목을 선택하면 원본 URL, 재생 시간, 해상도, 용량, 파일명, 메타데이터 상태와 UUID를 확인할 수 있습니다.

저장된 영상과 오디오는 다시 내려받지 않고 **Preview**로 재생할 수 있습니다. 실패 작업 재시도, 브라우저 다운로드, 기록만 삭제, 실제 파일 삭제 작업도 제공됩니다.

## 마운트된 파일

`/downfolder`에 기존 파일이 있으면 자동으로 탐색합니다. 저장된 다운로드 기록이 없는 파일은 **Mounted**, **No metadata**로 표시됩니다. 미리보기, 브라우저 다운로드와 삭제는 가능하지만 원본 URL이나 화질 정보는 복원할 수 없습니다.

기록만 모두 지워도 실제 파일은 삭제되지 않습니다. 남겨진 파일은 다시 마운트 파일로 표시됩니다.

## 컨테이너 업데이트

```shell
docker compose pull
docker compose up -d
```

`latest`와 `26.0713` 이미지는 `linux/amd64`, `linux/arm64`를 지원합니다.

휴대폰 공유 설정은 [모바일 공유 설정](../mobile/ko.html)을 참고하세요.
